Welcome to the TDS Statistics Editor!

A community made tool for creating and editing Tower Defense Simulator tower stats, duh.

Getting Started

  1. Select a tower from the search bar in the header
  2. Tinker with the statistic values!
  3. Export your changes as JSON or Wikitable
  4. Share it to everyone! On Discord, X, Reddit, or even our Database!

    5. Pro Tip: Seeing an outdated table? Try resetting to default or clear all data! You might be on an older cache.

Key Features

  • Edit tower stats and attributes
  • Create custom towers using the clone tool
  • Delta view to see your changes highlighted
  • Generate wikitable code for the TDS Wiki
  • Upload your changes to the Database

Understanding the Interface

Sidebar
  • Upgrade Viewer - See tower upgrades, abilities, and edit its summary
  • Property Viewer - Toggle which stats will appear in the table
  • Boost View - Add various buffs or debuffs to see their effects
Toolbar
  • Sidebar Toggle - Compresses the sidebar and hide it
  • Community Poll - View the community poll that you might have missed
  • Show Deltas - Highlight changes made to values
  • Clone - Copy an existing tower and make a new entry
  • Settings - Set your preferences
Navigation Bar
  • Table - Edit stats with a user friendly interface
  • Wikitable - Generate tables to be used anywhere on the wiki
  • JSON - View, export, and import JSONs
  • About - A description, disclaimer, and update log of the site
